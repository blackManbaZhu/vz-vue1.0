# To start

 * 使用了webpack2 模块化管理
 * 使用vuetify UI组件
 * 修改了导航栏
 * 使用了vue-cli脚手架 [vue-cli](https://github.com/vuejs/vue-cli)

``` bash
# install dependencies 下载依赖
cnpm install

# serve with hot reload at localhost:8081 运行项目
npm run dev

# build for production with minification 
npm run build

# author
yongji zhu

```
